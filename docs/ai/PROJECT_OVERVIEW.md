# Обзор проекта

## Назначение

«Суздальская IT Долина» — русскоязычный публичный лендинг технологической резиденции под Суздалем для инженеров, исследователей, IT-команд, технологических проектов и инвесторов с длинным горизонтом.

## Состав

- Один frontend-сервис.
- Основная страница лендинга по `/`.
- Отдельное представление команды по `/team`.
- Статические изображения и SEO-файлы в `public/`.

## Основные пользовательские сценарии

- Изучить концепцию, направления технологий и план развития Долины.
- Ознакомиться с предложениями для резидентов и инвесторов.
- Посмотреть основателей проекта.
- Перейти к контакту через email или, при настройке, Telegram.

## Границы текущей реализации

В репозитории нет отдельного backend-сервиса, базы данных и административного интерфейса. Node-процесс frontend-сервиса содержит единственную серверную точку `POST /api/contact` для безопасной доставки заявок в Telegram.
